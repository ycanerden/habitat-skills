# Founder Context

## Product
**Name:** DevMetrics
**One-liner:** Real-time analytics for GitHub repositories
**What it does:** Tracks pull request velocity, code review time, and contributor activity. Helps engineering managers understand team productivity.

## Target Audience
- Engineering managers at startups (10-50 person teams)
- Technical founders managing small dev teams
- DevOps teams who need visibility into development workflows

## Stage
- MVP built and tested with 50 beta users
- Planning public launch in 2 weeks
- Currently free beta, will introduce paid tiers at launch

- Haystack (expensive, $50+/user/month)
